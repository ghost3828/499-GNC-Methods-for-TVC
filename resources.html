Reference Library

Every reference for the study, organized by subject. Start here marks the recommended first step in each area; Shared folder marks material in \USAFA Astro - GNC TVC Study\ rather than on the open web.

Motivation & Worked Examples

  • Start here BPS.space — Scout F, Engineering Cut The whole mission in one video. Simulation section at 18:36; TVC hardware section at 1:11:15.
  • BPS.space — Modeling a Thrust Vectored Rocket in Simulink
  • MathWorks — Modeling a TVC'd Rocket in Simulink Reference architecture from MathWorks; compare against your own build rather than copying.
  • TVC Rocket PID Tuning in Simulink — Ep. 3
  • Shared folder Example TVC simulation code — 1.References\2. Example Code Instructor's caveat: no guarantee these examples are correct or follow best practices. Read critically.

Attitude Kinematics & Dynamics

  • Start here DCM Tutorial — An Introduction to Orientation Kinematics (starlino)
  • Shared folder AFIT MECH-632 slides — attitude parametrizations, kinematics, Euler's equations
  • Wie, Bong. Space Vehicle Dynamics and Control The rigorous backstop for Phase 1. A space flight dynamics course (e.g., Astro 321) covers the same ground.

Modeling & Simulation

  • Shared folder MATLAB Review slides
  • A Simple MEMS Gyro Model Using MATLAB/Simulink
  • thrustcurve.org Measured thrust curves for hobby motors — source for the sim's thrust lookup table.

Controls

  • Start here Understanding PID Control — Brian Douglas, MATLAB Tech Talks
  • Arduino PID Library — Brett Beauregard The reference implementation of "Improving the Beginner's PID" — the original blog series is unreliable, but its lessons (sample time, derivative-on-measurement, integrator clamping) are baked directly into this widely used, actively maintained library. Read the source and comments.
  • Anti-Windup Control Using a PID Controller — MathWorks documentation
  • Brian Douglas — Control System Lectures
  • Steve Brunton — eigensteve Control Bootcamp and beyond — the deeper theory behind the stretch goals (LQR, state-space methods).
  • Franklin, Powell & Emami-Naeini. Feedback Control of Dynamic Systems A rigorous backstop for Phase 2, the controls counterpart to Wie for Phase 1 — root locus, frequency response, and state-space design beyond PID.

Navigation & Estimation

  • Start here Understanding Sensor Fusion and Tracking — MATLAB Tech Talks
  • A Practical Approach to Kalman Filter and How to Implement It — TKJ Electronics
  • Kalman and Bayesian Filters in Python — Roger R. Labbe Jr. Free, interactive (Jupyter) book. Goes well beyond the TKJ tutorial if the Day 9/13 Kalman filter stretch goal grabs you.

Embedded & Hardware

  • Start here Arduino — Blink Without Delay The single most important embedded pattern in this course: non-blocking, fixed-rate loops.
  • arduino.cc — language and library reference
  • Teensyduino The Arduino IDE add-on that targets the Teensy 4.0. Install this before Day 11.
  • Adafruit BNO085 guide and Adafruit BMP390 guide — wiring, libraries, and example sketches for Day 12
  • How Servo Motors Work & How to Interface Them with Arduino
  • Orientation — Arduino quaternion library Untested by the instructor. Verify against your own MATLAB results before trusting it in the loop.
  • Thingiverse — open-source TVC mount designs
  • Blue Bird servos — BPS.space's TVC servo choice
  • Full parts list with roles and wiring notes → Hardware page

Glossary

GNCGuidance, Navigation, and Control
TVCThrust Vector Control — steering by gimbaling the motor
COTSCommercial off-the-shelf
DCMDirection Cosine Matrix (rotation matrix)
EOMEquations of motion
3DOF / 6DOFDegrees of freedom simulated: planar (x, z, pitch) vs. full 3D translation + rotation
IMUInertial Measurement Unit — 3-axis gyro + 3-axis accelerometer
AGLAbove ground level
ZOHZero-order hold — models sampled (discrete-time) signals held between updates
I²C / UARTSerial communication protocols: shared sensor bus / point-to-point link
ODROutput data rate (of a sensor)
MRMilestone review
